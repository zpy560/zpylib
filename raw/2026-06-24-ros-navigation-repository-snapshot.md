## README facts

- README title: ROS Navigation Stack.
- It is a 2D navigation stack that takes odometry, sensor streams and a goal pose.
- It outputs safe velocity commands sent to a mobile base.
- Related stacks include `navigation_msgs`, `navigation_tutorials` and `navigation_experimental`.
- Discussion is directed to the `ros-sig-navigation` mailing list.

## Interpretation boundary

- This is the ROS1 navigation stack, not ROS2 Navigation2.
- It is useful for understanding the historical `move_base` architecture and comparing ROS1/ROS2 navigation evolution.
