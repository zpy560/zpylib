## 代码形态

典型结构：

```text
固定容量宏
→ 全局数组和结构体
→ 算法函数
→ main() 从 stdin 读取
→ stdout 输出结果
```

代码特征：

- 103 个文件全部包含 `main()`。
- 99 个文件使用 `using namespace std`。
- 不使用 `bits/stdc++.h`，多为 C 标准输入输出。
- 只有少量文件包含说明性注释。
- 多数容量通过宏固定，例如顶点数、边数和字符集大小。
- 输入格式主要由源码推断，README 没有逐算法输入输出和复杂度说明。
- 20 个文件使用 `new`，只有 2 个文件出现 `delete`。
- 部分实现与具体竞赛题绑定，例如 Link-Cut Tree 文件直接解决 `[HNOI2010] 弹飞绵羊`。

这些代码更接近“可提交程序模板”，而不是可链接的算法组件。
