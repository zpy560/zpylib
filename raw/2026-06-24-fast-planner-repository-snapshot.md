## README facts

- Fast-Planner aims to enable quadrotor fast flight in complex unknown environments.
- It contains a rich set of carefully designed planning algorithms.
- It provides framework and algorithms that support projects including ego-planner, FUEL and RACER.
- It has been tested on Ubuntu 18.04 ROS Melodic and Ubuntu 20.04 ROS Noetic.
- README installation begins with installing `nlopt v2.7.1`.

## Interpretation boundary

- Fast-Planner is a quadrotor planning framework, not a ground vehicle planning stack.
- It is useful for B-spline/topological/local replanning ideas in unknown 3D environments.
