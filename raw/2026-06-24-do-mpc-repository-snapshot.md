## README facts

- do-mpc is a comprehensive open-source toolbox for robust model predictive control and moving horizon estimation.
- It supports nonlinear systems, uncertainty handling and time discretization.
- Features include nonlinear/economic MPC, DAE support, orthogonal collocation, robust multi-stage MPC, MHE and modular design.
- It is Python based and works on Python 3.x.
- Documentation is at `www.do-mpc.com`.
- README citation mentions a 2023 Control Engineering Practice paper.

## Interpretation boundary

- do-mpc is a high-level Python MPC/MHE toolbox, not a hard real-time embedded control runtime by default.
- It is useful for formulation, simulation and research prototypes.
