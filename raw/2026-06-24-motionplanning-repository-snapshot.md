## README facts

- The repository implements common motion planners used on autonomous vehicles.
- Listed planners include Hybrid A*, Frenet Optimal Trajectory and incomplete H-OBCA.
- Listed controllers include Pure Pursuit + PID, rear-wheel feedback + PID, Stanley + PID, LQR + PID and Linear MPC.
- Requirements include Python 3.6+, SciPy, cvxpy, Reeds-Shepp Curves and pycubicspline.
- Vehicle models include a simple car model and a car-pulling-trailers model.

## Interpretation boundary

- This is an autonomous-driving-oriented demo repository, not a full planning/control stack.
- It is most useful for comparing planner/controller behavior in compact Python examples.
