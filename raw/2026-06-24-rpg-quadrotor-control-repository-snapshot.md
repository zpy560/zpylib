## README facts

- README title: RPG Quadrotor Control.
- The repository provides ROS packages.
- It is research code and disclaims fitness for a particular purpose.
- It contains a complete framework for flying quadrotors based on control algorithms developed by the Robotics and Perception Group.
- It provides an interface to RotorS Gazebo plugins, a simple trajectory generation library, gamepad command utilities, battery voltage calibration routines and flight controller communication for FPV racing.
- The theory is summarized in a document and RA-L 2017/2018 papers.

## Interpretation boundary

- This is quadrotor control research code, not a general planning stack.
- It is useful for aggressive quadrotor tracking and control architecture references.
