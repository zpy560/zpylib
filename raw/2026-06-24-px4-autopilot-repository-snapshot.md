## README facts

- README tagline: The autopilot stack the industry builds on.
- PX4 is an open-source autopilot stack for drones and unmanned vehicles.
- It supports multirotors, fixed-wing, VTOL, rovers and other experimental platforms.
- It runs on NuttX, Linux and macOS.
- License is BSD 3-Clause.
- PX4 architecture uses uORB and DDS-compatible publish/subscribe middleware.
- It provides MAVLink and DDS/ROS 2 integration, SITL simulation, HIL testing and log analysis tools.
- PX4 is hosted under Dronecode Foundation, part of the Linux Foundation.

## Interpretation boundary

- PX4 is a full autopilot stack, not a single control algorithm repository.
- It is highly relevant for flight control architecture, estimator/control modules and robotics middleware integration.
