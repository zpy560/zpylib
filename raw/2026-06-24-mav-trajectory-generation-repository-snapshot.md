## README facts

- The repository contains tools for polynomial trajectory generation and optimization.
- The methods are especially suitable for rotary-wing micro aerial vehicles.
- README cites Richter, Bry and Roy on polynomial trajectory planning for aggressive quadrotor flight.
- README also cites work on real-time visual-inertial mapping, re-localization and planning onboard MAVs.
- It is a ROS catkin package with installation instructions for ROS Indigo/Kinetic/Melodic.

## Interpretation boundary

- This is a polynomial trajectory generation utility for MAVs, not a full autonomy stack.
- It is useful after waypoints or vertices are selected and a smooth trajectory is needed.
