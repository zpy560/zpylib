## README facts

- The project describes itself as Python code and a textbook for robotics algorithms.
- It has CI badges for Linux, macOS and Windows.
- Content spans localization, mapping, SLAM, path planning, path tracking, arm navigation, aerial navigation and bipedal examples.
- Planning algorithms listed include Dijkstra, A*, D*, D* Lite, potential field, coverage planning, PSO, state lattice, PRM, RRT/RRT*, Reeds-Shepp, quintic polynomial, LQR planning and Frenet optimal trajectory.
- Tracking/control examples include move-to-pose, Stanley, rear-wheel feedback, LQR speed/steering, MPC speed/steering and nonlinear MPC with C-GMRES.

## Interpretation boundary

- This is a learning and reference implementation repository, not a production robotics framework.
- It is a high-value algorithm index for understanding planning/control ideas before reading C++ or ROS integrations.
