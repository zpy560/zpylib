## README facts

- ACADO Toolkit is described as a toolkit for automatic control and dynamic optimization.
- It provides algorithms for direct optimal control, model predictive control, state and parameter estimation and robust optimization.
- It is implemented as self-contained C++ code and includes a MATLAB interface.
- The object-oriented design is intended to couple existing optimization packages and allow user-written optimization routines.

## Local interpretation

ACADO belongs to the optimal-control toolchain family. It is relevant historically and conceptually, especially for NMPC/OCP modeling, but new projects should compare it with acados, CasADi, do-mpc and OCS2 before choosing it.
