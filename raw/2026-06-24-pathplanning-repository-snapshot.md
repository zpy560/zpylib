## README facts

- The repository implements common path planning algorithms used in robotics.
- It covers search-based algorithms and sampling-based algorithms.
- Search-based algorithms listed include BFS, DFS, Best-First, Dijkstra, A*, Bidirectional A*, ARA*, LRTA*, RTAA*, LPA*, D*, D* Lite and Anytime D*.
- Sampling-based algorithms listed include RRT, RRT-Connect, Extended-RRT, Dynamic-RRT, RRT*, Informed RRT*, RRT* Smart, Anytime RRT*, Closed-Loop RRT*, Spline-RRT*, FMT* and BIT*.
- README includes animations and related paper links.

## Interpretation boundary

- This is a path planning algorithm demo repository, not a full navigation stack.
- It is useful for comparing graph search and sampling-based planners in compact examples.
