# F1TENTH planning：小车竞速规划算法库

## 一句话结论

F1TENTH planning 适合学习自动驾驶小车竞速场景中的规划控制算法，价值在轻量、实验性和闭环验证；不能直接外推到量产乘用车规划控制。

## 原文要点

- README 标题为 `f1tenth_planning`。
- 仓库属于 F1TENTH 生态。
- 当前快照分支为 `main`，HEAD 为 `70f0b6c`。

## 我的判断

F1TENTH 是连接教学、竞赛和自动驾驶算法验证的好场景。它比纯算法库更接近闭环系统，但尺度、动力学、传感器和安全要求与真实车辆不同。
