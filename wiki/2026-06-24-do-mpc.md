# do-mpc

## 基本信息

- 类型：Python MPC/MHE 工具箱
- 官方仓库：https://github.com/do-mpc/do-mpc
- 文档：https://www.do-mpc.com
- 快照分支：`master`
- 快照提交：`90e0d7ada888a4cc75ea3769214de5105764c640`
- 快照日期：2026-06-24

## 当前结论

do-mpc 是 MPC 学习、建模和仿真的好入口，尤其适合 nonlinear MPC、robust MPC 和 MHE 原型。实时部署应再下探到求解器和系统接口。

## 核心依据

- README 明确定位为 robust MPC 和 MHE toolbox。
- 支持 DAE、orthogonal collocation、multi-stage MPC。
- Python 工具箱，文档和论文引用路径明确。

## 不同观点与冲突

- 作为 MPC 原型工具：优先级高。
- 作为嵌入式实时控制器：需要额外验证。
- 作为运动规划框架：不适用。
